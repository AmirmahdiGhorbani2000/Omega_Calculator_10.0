# 🧮 Omega Calculator 10.0

A modular engineering calculator with 69 physics formulas, 118 periodic table elements, polynomial solvers up to degree 10, matrix operations, numerical calculus, and a bilingual interface.

## Author

**Amirmahdi Ghorbani**  
[github.com/AmirmahdiGhorbani2000](https://github.com/AmirmahdiGhorbani2000)

## Features

- 69 physics formulas across 8 categories (Kinematics, Dynamics, Energy, Fluids, Electricity, Thermodynamics, Optics, Modern Physics)
- Complete periodic table with all 118 elements and 11 properties each
- Polynomial equation solver from degree 1 to 10 using Newton-Raphson and Cardano methods
- Matrix operations: addition, multiplication, determinant, and inverse
- Numerical calculus: derivative via central difference and integral via Simpson's rule
- Statistics: mean, median, mode, and 10 probability distributions
- Geometry: perimeter, area, volume, Pythagorean theorem, trigonometric functions
- Unit conversion for length, temperature, and weight
- Number theory: prime detection, GCD/LCM, Fibonacci, Pascal's triangle, number bases
- Fully bilingual: complete English and Persian interfaces

## Architecture

The project is organized into 10 independent modules:

- main.py — Entry point with language selection
- math_base.py — Mathematical utilities and dynamic function compilation
- equations.py — Polynomial solvers for degrees 1 through 10
- calculus.py — Numerical derivative and Simpson integration
- matrix_ops.py — Matrix algebra operations
- constants.py — 17 fundamental physical constants
- formulas.py — 69 physics formulas across 8 subfields
- periodic_table.py — Complete 118-element database
- english.py — English-language interface
- persian.py — Persian-language interface
